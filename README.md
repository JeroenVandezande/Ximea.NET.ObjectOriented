# Ximea.NET.ObjectOriented

An object-oriented .NET wrapper for Ximea cameras, providing a clean and type-safe interface for camera control and image acquisition.

## Overview

Ximea.NET.ObjectOriented is a modern .NET library that provides an object-oriented implementation of the Ximea Camera API. It builds upon the xiAPI.NET.CrossPlatform package and implements the CameraInterface for standardized camera operations.

## Features

- Object-oriented design for intuitive camera control
- Cross-platform support through xiAPI.NET.CrossPlatform
- Implements standard camera interface for compatibility
- Type-safe API for camera operations
- Support for .NET 8.0 and .NET 9.0

## Installation

Install via NuGet Package Manager:

```bash
dotnet add package Ximea.NET.ObjectOriented
```
